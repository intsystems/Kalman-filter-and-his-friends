Kalman Filter

The standard (linear) Kalman filter for systems described by linear state-space equations with Gaussian noise. Also defines the BaseFilter interface shared by all filter variants.

class kalman.filters.SPDParameter(init_matrix: Tensor)[source]

Bases: Module

Symmetric Positive Definite matrix parameterized via Cholesky factor. Stores a lower-triangular L such that the matrix is L @ L^T. This guarantees the output is always SPD during optimization.
